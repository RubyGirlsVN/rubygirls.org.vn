Thuật ngữ

app / application

ứng dụng: sản phẩm phát triển cho một nền tảng (như di động, web). VD: ứng dụng Android, ứng dụng web.

authentication

xác thực: quá trình xác nhận danh tính của người dùng, thường là email / username và mật khẩu.

authorization

xác nhận quyền: quá trình xác nhận các quyền của người dùng. VD: xác nhận xem người dùng có quyền truy cập vào một phần nhất định của trang web hay không.

Bootstrap

HTML, CSS và JavaScript framework phát triển bởi Twitter cho phép các nhà phát triển xây dựng một giao diện web hiện đại một cách nhanh chóng và tiện lợi.

build (continuous integration)

một bản sao chạy được của ứng dụng được tạo ra với mục đích kiểm thử, triển khai ứng dụng.

code

mã / viết mã

comment (trong code)

chú thích

commit

cam kết các thay đổi bạn thực hiện vào trong lịch sử của repository.

continuous deployment

triển khai liên tục: liên tục triển khai phiên bản mới nhất của dự án vào sử dụng.

dashboard

bảng điều khiển: thường là trang bắt đầu của các giao diện quản lý, nơi cung cấp cho bạn cái nhìn tổng quan về các thông số quan trọng cũng như đường dẫn tới các tài nguyên liên quan.

database

cơ sở dữ liệu.

dependencies

các thư viện, package (gói phần mềm) mà một phần mềm / thư viện cần để có thể hoạt động.

deploy / deployment

triển khai ứng dụng: hành động / việc đưa ứng dụng (hoặc những thay đổi mới trong ứng dụng) vào sử dụng.

download

tải về

dyno

đơn vị server của Heroku

framework

bộ khung cơ bản: gồm các thư viện thường dùng, các công cụ tiện ích giúp cho việc phát triển trở lên thuận lợi và nhanh chóng nhờ việc bỏ qua những công việc mang tính chất “phát minh lại bánh xe”. VD: Ruby on Rails là một framework xây dựng trên ngôn ngữ Ruby giúp cho việc xây dựng các ứng dụng web trở lên nhanh chóng và dễ dàng hơn.

ephemeral storage

lưu trữ không bền vững: các tệp tin mới được tạo trong quá trình chạy sẽ không được giữ lại khi server dừng hoặc khởi động lại

field (database)

trường: các cơ sở dữ liệu cổ điển lưu dữ liệu dưới dạng bảng, một cách đơn giản thì trường là tên của các cột trong bảng

form (HTML)

biểu mẫu: giống như các biểu mẫu trên giấy, các biểu mẫu trên web cũng cùng mục đích thu thập các thông người dùng nhập vào với nhiều mục đích khác nhau, như đăng nhập, yêu cầu hỗ trợ, mua hàng…

gem

thư viện đã được đóng gói theo một tiêu chuẩn trong Ruby

generator script (Rails)

các lệnh cho phép tạo ra một loạt tệp tin / code cần thiết cho một số tác vụ thường dùng, như tạo model mới, tạo controller mới

Gravatar

một dịch vụ cho phép liên kết email của bạn với một ảnh đại diện, giúp bạn có một ảnh đại diện thống nhất và tự động khi sử dụng nhiều dịch vụ web

grid layout

bố cục dạng lưới: bố cục được căn chỉnh dựa trên một nên dạng lưới vuông, giúp cho các giao diện trông cân đối, hài hòa

helper (Rails)

module cho phép tạo các đoạn code có thể được sử dụng ở nhiều chỗ, tránh việc lặp lại các đoạn code dài

Heroku

Dịch vụ đám mây cung cấp giải pháp nhanh gọn để deploy các ứng dụng (như các ứng dụng làm trên RoR)

Heroku Toolbelt

phần mềm của Heroku cài trên hệ điều hành để tương tác với các dịch vụ của Heroku thông qua terminal

JavaScript

ngôn ngữ kịch bản có thể chạy trên cả server và trên trình duyệt, làm cho các ứng dụng web có độ tương tác cao hơn

layout

bố cục

log / logging

quá trình ghi lại các sự kiện, biến cố của một phần mềm / ứng dụng vào một tệp tin, giúp cho việc theo dõi, quản lý ứng dụng thuận lợi hơn, đặc biệt là khi có lỗi xảy ra

migrate database (Rails)

dịch chuyển cơ sở dữ liệu: quá trình đồng nhất các thay đổi trong cấu trúc model vào cơ sở dữ liệu

navigation

Thanh điều hướng: tương tự như menu, giúp việc dịch chuyển giữa các trang khac snhau của một website thuận lợi hơn.

output

là hệ quả / kết quả / quá trình chạy của các lệnh được in ra màn hình

package manager (Linux)

phần mềm quản lí việc cài đặt / gỡ cài đặt các thư viện / phần mềm / ứng dụng khác trong Linux.

PostgreSQL

một cơ sở dữ liệu mã nguồn mở mạnh mẽ, được dùng làm cơ sở dữ liệu mặc định cho các ứng dụng cài đặt trên Heroku

push code

đẩy mã: lên các kho lưu trữ (repository) ở nơi khác, như GitHub, Heroku, v.v.

RDBMS (Relational Database Management System)

hệ thống quản lý cơ sở dữ liệu dạng relational (có liên quan): dữ liệu được lưu trữ trong các “bảng” và các bảng này “có liên quan” với nhau bởi các trường lưu trữ thông tin nhận dạng của bảng khác (khóa ngoại - foreign key). VD: MySQL, PostgreSQL

refactor

quá trình tối ưu lại các đoạn code đã viết (đẹp hơn, dễ hiểu hơn, xử lý tốt hơn, v.v.)

repository / repo

kho lưu trữ: code và lịch sử thay đổi của code

route(s)

định nghĩa các địa chỉ nào sẽ dẫn tới các tài nguyên nào

Rubyist

cách những lập trình viên Ruby (và yêu mến Ruby) tự gọi mình

specs / specification

đặc tả kĩ thuật

static assets

tài nguyên tĩnh

stylesheet

tệp tin CSS

terminal

công cụ sử dụng dòng lệnh để tương tác (trên window là Command Prompt)

test / testing

kiểm thử

test case

một trường hợp kiểm thử cụ thể với các điều kiện đầu vào và kết quả mong muốn

thumbnail

phiên bản với kích thước thu nhỏ của một hình ảnh, mục đích chính là giảm kích thước của dữ liệu web khi không cần dùng ảnh với kích thước gốc

upload

tải lên

Version Control Systems (VCS)

Hệ thống quản lý phiên bản, dùng để quản lý các thay đổi trong mã nguồn của phần mềm.
